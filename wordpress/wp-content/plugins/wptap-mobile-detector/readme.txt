WPtap is a WordPress specialist in developing mobile-friendly themes/plugins that can instantly turn your WordPress site into a web-application experience. Our themes currently support iPhone, Android, and Touch-based Blackberry series.

== Description ==

This plugin automatically detects the type of mobile browser that you site is viewed from, and activates the mobile theme you have chosen for it. User can install multiple mobile themes and link it to different mobile browsers for best performance. If you have a separate WAP or mobile website, this detector also allows you to redirect your mobile traffic to the WAP/mobile site.

Features:
1.	Automatically detects mobile browsers & mobile user agents
2.	Automatically detects installed PC and mobile themes
3.	User can link different mobile themes to different mobile devices for best performance.
4.	Option to redirect mobile traffic to a separate WAP or mobile site
5.	Complete mobile device listing (iPhone, iPod, Android, Blackberry, etc.) User can also add devices to this list.

Note: Mobile Detector does NOT run together with WPtap plugin, it has already been integrated into WPtap themes/plugin.

== Frequently Asked Questions ==

= Why do I need WPtap Mobile Detector?

1. If you have a mobile site or WAP site and wish to direct mobile traffic automatically to it.
2. If you are using a WPtap mobile theme and wish to automatically detect mobile devices and switch themes.
3. If you are using other mobile themes and wish to activate them for lower-end phones such as Symbian etc.
